# SDK Exploration

This directory contains exploration documents for exposing a Rust SDK
from the Torrust Tracker Deployer, enabling other projects and AI agents
to programmatically interact with the deployer.

| [Discussion](discussion.md)                      | Goals, rationale, design decisions, and open questions      |
| [Phase 1: PoC](phase-1-poc.md)                   | Proof-of-concept plan with exposed types and example code   |
| [Phase 2: Improvements](phase-2-improvements.md) | Post-PoC improvements ordered from simplest to most complex |
